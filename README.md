# Attractive College Website

Full-stack college website with frontend + backend, using:
- HTML, CSS, JavaScript, jQuery
- Node.js, Express, MongoDB (Mongoose)
- CRUD for Students, Courses, Faculty, Attendance
- Simple JWT auth for Admin/Faculty/Student

## Quick start (local)
1. Install Node dependencies:
   ```
   npm install
   ```
2. Create a `.env` file based on `.env.example`.
3. Start MongoDB (e.g., mongod).
4. Run server:
   ```
   npm run dev
   ```
5. Open `frontend/index.html` in your browser (or serve `frontend/` with a static host).

## Deploy
- Backend: Render / Heroku / Railway
- Frontend: Vercel / Netlify (or serve with backend static)
